Projekt semestralny z czwartego semestru jest obecnie w produkcji.
Będzie to aplikacja piłkarska w której będzie można sprawdzić aktualną
sytuację w kilku najlepszych europejskich ligach piłkarskich.

Aplikacja wykorzysta API do pobierania zawsze aktualnych danych 
i GUI do wyświetlania ich.

Język: C++
